Portfolio

  • Bus Route Planning System
    My group and I used Java in Eclipse to create a bus route planning system that would calculate the distance to a certain location, the amount of gas to be used on the trip. stops along the way, and if a gas station would be necessary to reach the destination. Skills involved were Java programming, database creation and management, GUI and console implementation, and communication between group members.

  • Joke Generator
    My group and I created a joke generator that would allow a user to pick from various categories of joke types and other selections and provide a joke and punchline for the user. Skills involved were Java programming, creation and implementation of audio files, creation and management of a small database, and GUI implementation.
  • How-To Website
    I created a website that walked a user through the downloading of a videogame known as Clone Hero on their Mac device. Skills involved were HTML and CSS programming, much like in CSCI 225.
